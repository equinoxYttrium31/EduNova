# EduNova
**An Educational Game Website for Interactive Learning**

EduNova is an interactive platform designed to enhance learning through engaging educational games and quizzes. It allows students to learn various subjects in a fun and dynamic way, making education more enjoyable and accessible. The platform includes a range of games and quizzes for different subjects, user authentication via popular methods like Google and Facebook, and a user-friendly interface.

## Project Description

EduNova is a web application aimed at helping students of all ages enhance their knowledge through gamified learning. By offering quizzes and games in various subjects, EduNova provides a fun and engaging way to learn. Users can log in to track their progress, challenge their friends, and access personalized learning content.

---

## Features

- **Educational Games & Quizzes**: Fun games and quizzes for different subjects that make learning more engaging.
- **User Authentication**: Easy login via Google and Facebook accounts for seamless access.
- **Progress Tracking**: Users can track their performance over time.
- **Responsive Design**: The website is fully responsive and works well on both desktop and mobile devices.
- **Leaderboard**: See how you rank against other users based on quiz performance.

## Installation

To run EduNova locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/equinoxYttrium31/EduNova.git
2. **Navigate into the project directory**:
    cd EduNova
3. **Install dependencies: Make sure you have Node.js installed, then run**:
    npm install
4. **Start the development server**:
    npm run dev

Your project should now be running at http://localhost:5173
